post /event/new
大会を開催する
送る情報  大会の情報（大会名、ボードゲームの種類、主催者、日時、場所、モチベーション）
返される情報 なし

get /event/games
ボードゲーム種類とモチベーションを選択し、大会の一覧を表示
送る情報  ボードゲームの種類と、モチベーション
返される情報  参加可能な全ての大会のIDと大会名

get /event/:id
指定したイベントの詳細を表示
送る情報  選択した大会のID
返される情報  大会の情報

post /event/attend
大会に参加する
送る情報  ユーザーID	
返される情報  なし

get /event/users
大会参加者の一覧を表示
送る情報  大会のID
返される情報  参加予定のユーザーID、ユーザー名
